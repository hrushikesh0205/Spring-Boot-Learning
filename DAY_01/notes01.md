## Theory
- Spring is a Java framework used to build enterprise applications.
- Spring Boot is an extension of Spring that reduces configuration.
- Dependency Injection allows Spring to provide required objects.
- Inversion of Control means Spring manages object creation.
- AOP separates common logic from business logic.
